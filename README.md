# Asistencia

## 🔭 ¿Qué es?
**Asistencia** es un proyecto que busca facilitar el registro de estudiantes en sus clases mediante el escaneo de codigos QR.

## 💡 ¿Cómo funciona?
El sistema consiste en un sitio web que permite generar el codigo QR, este QR permitirá a los estudiantes; tras un proceso de registro, escanearlo y que quede constancia de su asistencia a clase.

## 🚀 Tecnologias 
Para este proyecto se utilizo:

### Front-End
- HTML
- CSS
- JS

### Back-End
- PHP
- MySQL (Base de datos)
